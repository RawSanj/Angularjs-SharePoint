    Help - Get aware of Issue Tracker Tool!


    This section deals with the Features of Issue Tracker Tools and what can be achieved with it.

    Home Page UI -

    Active Issues : The first page is loaded with the Active Issues.
    Click on Issues Refresh icon to reload the Page and refresf the Result with Active Issues.

    Release : Select any Release from the Release dropdown to fetch the Issues repoerted for that Release.

    Search by Tittle: Type the Title of the Issues to filter the results displayed in the table.

    Search Any Issue by Id: Type the Issue Id of any Issue and click Submit to get the issue info.
    Note : This field looks for the Issue in Current as well as Archived issues. User can use this field to search for any historical Issues.

    Tools : Click on Tools dropdown to use tools such as Export to Excel, display graphs, etc.

    Issue Filter Panels : Click on the Filter links to filter the Issues by Severity, Component or Root Cause.
    Click Reset Filter to reset filter and display all issues in Result.
    Note: Only one filter is applied at one time.

    Tools -

    Export to XLS : Click on Export to XLS link to export all the Issues for the selected Release along with the Count of Issues as per Severity, Component and Root Cause.

    Visualize : Click on Visualize link to view Charts of Issues for the selected Release along with the Count of Issues as per Severity, Component and Root Cause.
    Each Charts in each Tabs are interactive, User can change the attributes to change the Chart and choose to download it various formats.
    User can also Save the Data the chart represents in CSV, XLSX or JSON format.
    Click on Annotate link to Draw or add Shapes, add Text over the Chart and then Download it in various formats.
